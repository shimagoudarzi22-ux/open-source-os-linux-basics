# Project Documentation

## Introduction
This project is designed to demonstrate basic Linux administration tasks and system automation as part of the Open Source Operating Systems course.

## Contents
- **System Monitoring Script:** A Bash script located in `src/` that checks CPU, Memory, and Disk usage.
- **Research:** Detailed information about Linux Kernel and Distributions in `researchs/`.
- **Project Info:** Metadata about the student and project version in `project_info.json`.

## How to use
To see the automation in action, run the script provided in the `src` folder.
